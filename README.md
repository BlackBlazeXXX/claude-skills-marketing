 ## ❓ FAQ
 
 **How do I use my own Brand Voice?**
 Fill out the provided `brand-voice.md` template and move it to your `~/.claude/skills/` directory. All skills will automatically find it.
 
 **Do I need to pay for anything?**
 No. This repo is 100% free and open source (MIT license).
 
 **Can I edit the skills?**
 Yes! Each `SKILL.md` is a plain text file — customize as much as you want.
