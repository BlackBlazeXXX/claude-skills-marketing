 ## 😩 The Problem
 
 Every time you need marketing content, you write the same long prompts:
 
 > *"Hey Claude, write me an Instagram post about my business, make it engaging, add hashtags, include a CTA, keep it under 200 words, use an emoji..."*
 
 **You type 50 words to get 1 post. Every. Single. Time.**
 
 ---
 
 ## ✅ The Solution
 
 ```bash
 /social-media-post Instagram fitness brand motivational
 ```
 
 **That's it. One command. Full professional output. Every time.**
 
 ---
 
 ## 🧠 The "Masterpiece" Upgrade (v2.0)
 
 We've upgraded every skill to a "Master" level with three major new features:
 
 1.  **🎭 Brand Voice Integration:** Create a `brand-voice.md` file in your skills directory. Every command will now automatically adapt to your specific brand tone, audience, and style. No more generic AI fluff.
 2.  **🧠 Psychology Breakdowns:** Every output now includes a "Why This Works" section, explaining the psychological triggers (Loss Aversion, Social Proof, Curiosity) used in the copy.
 3.  **🔄 3x Variations:** Every command now generates **3 distinct variations** (e.g., Professional, Edgy, Minimalist) so you can pick the perfect one for your goal.

 ## ❓ FAQ
 
 **How do I use my own Brand Voice?**
 Fill out the provided `brand-voice.md` template and move it to your `~/.claude/skills/` directory. All skills will automatically find it.
 
 **Do I need to pay for anything?**
 No. This repo is 100% free and open source (MIT license).
 
 **Can I edit the skills?**
 Yes! Each `SKILL.md` is a plain text file — customize as much as you want.
