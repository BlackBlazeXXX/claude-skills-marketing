# Contributing to Claude Marketing Skills 🤝

Thank you for helping grow this collection! Here's how to add a skill.

## Skill File Format

Create a folder: `skills/your-skill-name/SKILL.md`

```markdown
---
name: your-skill-name
description: One line — what this skill does and when Claude uses it automatically
argument-hint: "[arg1] [arg2] [arg3]"
---

Your instructions to Claude here...

## Instructions
1. Step one
2. Step two

## Output Format
### Section:
[what goes here]

## Rules
- Rule 1
- Rule 2
```

## Rules for Contributions

- ✅ Skill must be marketing related
- ✅ Must produce structured, complete output
- ✅ Name must be lowercase with hyphens
- ✅ Include at least 3 example use cases in PR description
- ❌ No duplicate skills
- ❌ No paid tool dependencies

## Steps

1. Fork this repo
2. Create your skill folder and SKILL.md
3. Test it in Claude Code
4. Open a Pull Request with examples of output

All contributors get credited in the README! 🙌
