## Instructions

Create scroll-stopping hooks using proven psychological triggers:

1. **Curiosity hooks** — open loops, incomplete info
2. **Contrarian hooks** — challenge common beliefs
3. **Number hooks** — specific data/lists
4. **Fear/urgency hooks** — what they might lose
5. **Desire hooks** — what they want to gain
6. **Story hooks** — personal/relatable openers
7. **Question hooks** — make them think

## Rules
- First 3 words are the most important — make them count
- Never start with "I" — start with power words or "You"
- Create an open loop — make them NEED to read more
- Be specific, not vague ("3x revenue" beats "grow your business")
- Test multiple hooks — what works is what matters
