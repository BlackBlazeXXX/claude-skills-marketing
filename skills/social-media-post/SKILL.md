Create a high-converting social media post based on: $ARGUMENTS

## Instructions

1. **Identify the platform** (Instagram / Twitter-X / LinkedIn / Facebook / TikTok)
2. **Write a scroll-stopping HOOK** (first line must grab attention instantly)
3. **Body content** — short, punchy, valuable
4. **Call to Action (CTA)** — tell them exactly what to do next
5. **Hashtags** — 5-10 relevant trending hashtags
6. **Emoji usage** — use emojis naturally to boost engagement

## Format Output As:

### Hook:
[attention-grabbing first line]

### Post:
[full post content]

### CTA:
[call to action]

### Hashtags:
[hashtags]

### Best Time to Post:
[suggest best posting time for this platform]

## Rules
- Keep Twitter/X under 280 characters
- LinkedIn posts = professional but human tone
- Instagram = visual storytelling + emotional
- Make it feel REAL, not robotic
- Always include ONE strong CTA
